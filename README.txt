=======
Макрос для Google Apps Script 
Очищает заданные диапазоны от данных, и вписывает новые, в зависимости от введенной даты. 
Проверяет исходную таблицу на пустые ячейки, для избежания смещения данных в целевой таблице.

=======
Macro for Google Apps Script
Clears specified ranges of data and enters new data based on the entered date.
Checks the source table for empty cells to avoid data misalignment in the target table.
